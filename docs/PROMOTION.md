## 产品定位

**AI 平台太多了，但 Nexior 真的不一样。**

市面上的 AI 工具往往是单一功能——要么只能聊天，要么只能画图。而 Nexior 把 **ChatGPT、Claude、DeepSeek、Gemini、Grok、Midjourney、Suno、Sora、Luma** 等 20+ 项顶级 AI 服务整合到了一个平台中，用户无需在多个 App 之间来回切换，一个入口满足所有需求。

更重要的是，Nexior 不仅是一个产品——它是一个 **开箱即用的 AI SaaS 方案**。你可以 Fork 源码，一键部署到 Vercel 或 Docker，拥有完全属于你的 AI 平台，自定义品牌、域名、功能、定价，然后通过内置的支付和分销系统变现。

## 商业模式 — 如何赚钱

### 分销返佣机制

Nexior 内置了完整的客户绑定和分销系统：

1. **客户绑定**：通过你的专属链接注册的用户，永久绑定为你的下级客户
2. **消费返佣**：下级客户每次消费（购买套餐），系统自动按比例返佣到你的账户
3. **多级返佣**：支持多级分销，级别越高返佣比例越大
4. **Cookie 追踪**：推荐链接有 7 天有效期，用户只要在 7 天内注册即绑定

### 收入结构

```
用户注册 → 绑定邀请关系 → 用户购买套餐 → 系统自动计算返佣
                                          ↓
                              邀请者账户增加收益
                                          ↓
                              联系客服提现（扣税后）
```

### 定价策略

- 积分制度：用户购买积分套餐
- 购买越多折扣越大
- 支持按量（次数）和按周期（时间）两种套餐
- VIP 折扣：持有 $ACE 代币可获额外折扣

## 部署指南 — 5 分钟搭建同款

### 方式一：Vercel 一键部署（推荐新手）

1. **Fork 仓库**
   - 访问 [https://github.com/AceDataCloud/Nexior](https://github.com/AceDataCloud/Nexior)
   - 点击 Fork

2. **部署到 Vercel**
   - 访问 [https://vercel.com](https://vercel.com)，用 GitHub 登录
   - 点击 Import → 选择 Nexior 仓库 → Deploy
   - 等待 1-2 分钟自动完成

3. **配置站点**
   - 自定义站点标题、Logo
   - 选择开启的 AI 功能
   - 配置分销参数

4. **绑定域名（可选）**
   - 在 Vercel 添加自定义域名
   - DNS 添加 A 记录指向 `76.76.21.21`
   - 等待生效

### 方式二：Docker 部署（适合有服务器）

```bash
# 构建镜像
docker build -t my-nexior .

# 启动服务
docker run -d -p 8000:80 my-nexior

# 访问 http://localhost:8000
```

配置 HTTPS：
```nginx
server {
    listen 443 ssl;
    server_name your-domain.com;
    ssl_certificate /path/to/cert.pem;
    ssl_certificate_key /path/to/key.key;
    location / {
        proxy_pass http://127.0.0.1:8000;
    }
}
```

### 方式三：本地开发

```bash
git clone https://github.com/AceDataCloud/Nexior.git
cd Nexior
npm install
cp .env.example .env
npm run dev
# 访问 http://localhost:8084
```

## 常见问题

**Q：Nexior 是免费的吗？**
A：Nexior 源码完全开源免费（MIT 许可证）。AI 服务按使用量付费，新用户有免费额度体验。

**Q：需要自己准备 AI 账号吗？**
A：不需要。Nexior 通过 AceDataCloud API 统一调用所有 AI 服务，你无需单独购买任何 AI 厂商的账号。

**Q：支持哪些支付方式？**
A：微信支付、支付宝、Stripe（国际信用卡/Apple Pay/Google Pay）、X402（Solana/Base USDC 加密货币支付）。

**Q：如何通过 Nexior 赚钱？**
A：部署你的平台 → 分享给用户 → 用户通过你的链接注册并消费 → 系统自动返佣到你的账户 → 联系客服提现。

**Q：可以自定义品牌吗？**
A：可以。支持自定义站点标题、Logo、Favicon、域名、SEO 配置，以及选择性开启/关闭各项功能。

**Q：支持多少种语言？**

**Q：有移动端 App 吗？**
A：有。基于 Capacitor 7 构建了 iOS 和 Android 原生 App，也支持网页端响应式访问。
